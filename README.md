# Blur to HD Image Enhancer

A Python-based AI image enhancement project developed using Google Colab. The project converts blurred or low-quality images into clearer high-definition (HD) images using image processing and deep learning techniques.

---

## Features

* Blur Image Enhancement
* HD Image Generation
* Image Upload Support
* AI-Based Image Processing
* High-Quality Image Output
* Google Colab Implementation

---

## Tech Stack

* Python
* Google Colab
* OpenCV
* Pillow
* Deep Learning Models

---

## Project Workflow

1. User uploads a blurred image in Google Colab.
2. The image is processed using AI and image enhancement techniques.
3. The model improves image clarity and quality.
4. Enhanced HD image is generated as output.
5. Users can download the processed image.

---

## Installation & Execution

### Open Google Colab

Upload the project notebook to Google Colab.

### Install Required Libraries

```bash id="a1b2c3"
pip install opencv-python pillow
```

### Run the Notebook

* Upload the input image.
* Execute all notebook cells.
* View and download the enhanced HD image.
